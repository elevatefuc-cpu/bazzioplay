# Bazzio Website (Static HTML)

This folder is **hosting-ready**. Upload everything inside `bazzio-site/` to your hosting (cPanel, GoDaddy, Netlify, etc.).

## Edit quickly
- Colors & fonts: `css/styles.css` (see `:root` palette).
- Logo: replace `assets/logo.jpg` with your logo (same name).
- Text: edit `.html` files in any editor (VS Code / Notepad).
- Phone/Email: search in `footer` and `contact.html`.
- Form: the demo just shows an alert. Connect to Google Forms or a backend to collect leads.

## Pages
- `index.html` — Home
- `programs.html` — Packages
- `about.html` — About
- `contact.html` — Contact
